Sehr geehrte Damen und Herren

Ein Transportunternehmen stellt den Antrag für die Umbenennung/Neueinführung der Haltestelle [[${properties.get(0).get('designationOfficial')}]] SLOID [[${properties.get(0).get('sloid')}]]. Sie dürfen im Rahmen der «Verordnung über geografische Namen (GeoNV)» über den Namen abstimmen. Bitte beantworten Sie die Anhörung bis zum [[${properties.get(0).get('endDate')}]] wie folgt:
  1. Aufrufen der Anhörung mit folgendem Link: [[${properties.get(0).get('url')}]]
  2. Entscheidung bezüglich des Namens der Haltestelle treffen
  3. Klicken Sie auf Enscheiden
  4. Geben Sie anschliessend die Mail-Adresse an, auf welche Sie diese Einladung erhalten haben
  5. Sie erhalten ein neues Mail mit dem PIN-Code für die Abstimmung
Bitte beantworten Sie die Anhörung in jedem Fall. Fehlende Antworten werden nach Ablauf der Frist als Einverständnis gewertet.
Weitere Informationen finden Sie unter https://www.öv-info.ch/de in der Beschreibung oder im Benutzerhandbuch.

Mesdames et Messieurs

Une entreprise de transport dépose une demande de changement de nom/d'introduction de l'arrêt [[${properties.get(0).get('designationOfficial')}]] SLOID [[${properties.get(0).get('sloid')}]]. Vous pouvez vous prononcer sur ce nom dans le cadre de l' « Ordonnance sur les noms géographiques (ONGéo)». Veuillez répondre comme suit à l'audition jusqu'au [[${properties.get(0).get('endDate')}]]:
  1. appeler l'audition en utilisant le lien suivant: [[${properties.get(0).get('url')}]]
  2. prendre une décision concernant le nom de l'arrêt
  3. cliquer sur Demander le code pin
  4. vous recevez un nouveau mail avec le code pin pour le vote
Veuillez répondre à l'audition dans tous les cas. L'absence de réponse sera considérée comme un accord à l'expiration du délai.
Vous trouverez de plus amples informations sur https://www.tp-info.ch/fr dans la description ou dans le manuel d'utilisation.
